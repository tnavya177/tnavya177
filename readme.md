## 🚀 Projects

### 💳 [Credit Card Fraud Detection](https://github.com/tnavya177/credit-card-fraud-detection)

> Python • Scikit-learn • XGBoost • SMOTE • Pandas • NumPy • Matplotlib • Seaborn • Streamlit

- Built a supervised ML classification model to detect fraudulent transactions from highly imbalanced datasets
- Applied **SMOTE** to handle class imbalance; performed EDA and feature engineering
- Trained and compared **Logistic Regression, Decision Tree, Random Forest, and XGBoost** classifiers
- Achieved **98% classification accuracy**; evaluated using Confusion Matrix, Precision, Recall, F1-Score, and ROC-AUC

### 💬 [Sentiment Analysis using NLP](https://github.com/tnavya177/sentiment-analysis)

> Python • NLTK • Scikit-learn • TF-IDF • CountVectorizer • Pandas • NumPy • Streamlit

- Developed an end-to-end NLP pipeline to classify customer reviews as **positive, negative, or neutral**
- Applied text preprocessing: tokenization, stopword removal, and stemming using **NLTK**
- Extracted features using **TF-IDF** and **CountVectorizer**
- Trained and compared **Naive Bayes, Logistic Regression, and SVM** classifiers
- Achieved **93% classification accuracy**; evaluated using Confusion Matrix, Precision, Recall, and F1-Score
